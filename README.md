Proyecto de análisis de datos enfocado en clientes de ConnectaTel.  
Se realizó limpieza, exploración y segmentación de usuarios para identificar patrones de uso, comportamiento y oportunidades de negocio.

---

## 🚀 Objetivos
- Limpiar y preparar datasets.
- Detectar valores faltantes y datos inconsistentes.
- Analizar comportamiento de usuarios.
- Identificar segmentos por edad y nivel de uso.
- Detectar outliers y patrones extremos.
- Generar recomendaciones de negocio.

---

## 🛠️ Tecnologías utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Análisis realizado
### Limpieza de datos
- Reemplazo de sentinels (`-999`, `?`)
- Conversión de fechas
- Manejo de valores nulos
- Validación de inconsistencias

### Exploratory Data Analysis (EDA)
- Estadísticas descriptivas
- Histogramas
- Boxplots
- Detección de outliers con IQR y Z-score

### Segmentación
- Segmentación por edad
- Segmentación por nivel de uso
- Clasificación de clientes

---

## 🔍 Principales hallazgos
- Existen usuarios con consumo extremadamente alto.
- Los usuarios de alto uso representan clientes valiosos.
- Los patrones de uso cambian según la edad y el plan contratado.
- Se detectaron oportunidades para crear planes personalizados.

---

## 💡 Recomendaciones
- Crear planes premium para usuarios intensivos.
- Diseñar promociones segmentadas por edad.
- Mejorar procesos de validación y calidad de datos.
